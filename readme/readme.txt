Plugin for CudaText.
It works in saved WikidPad documents, requires WikidPad lexer active.

By double-click over CamelCaseWord in document, plugin looks for file CamelCaseWord.wiki
in the current folder, and opens it, or suggests to create it.
It ignores \-escaped words like \CudaText.
